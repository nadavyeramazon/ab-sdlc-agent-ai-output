# Python Hello World Application

A simple Python application that demonstrates a "Hello World" program.

## Description

This application contains a basic Python script that:
- Prints "Hello, World!" to the console
- Demonstrates function usage
- Shows how to greet multiple people

## Usage

Run the application with:

```bash
python hello_world.py
```

## Requirements

- Python 3.6 or higher

## Features

- Simple greeting function
- Customizable name parameter
- Multiple greetings example
