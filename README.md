# Hello World Python Application

A simple Hello World application implemented in Python.

## Description

This is a basic Python application that demonstrates the classic "Hello, World!" program with additional features including:
- Basic hello world output
- Customizable greeting function
- Interactive user input for personalized greetings

## Requirements

- Python 3.6 or higher

## Installation

1. Clone the repository:
```bash
git clone https://github.com/nadavyeramazon/ab-sdlc-agent-ai-backend.git
cd ab-sdlc-agent-ai-backend
git checkout feautre/test-2
```

## Usage

### Run the application:

```bash
python hello_world.py
```

or

```bash
python3 hello_world.py
```

### Using the greeting function in your code:

```python
from hello_world import greet

# Basic usage
print(greet())  # Output: Hello, World!

# Custom greeting
print(greet("Alice"))  # Output: Hello, Alice!
```

## Features

- **Simple Output**: Prints "Hello, World!" to the console
- **Reusable Function**: The `greet()` function can be imported and used in other modules
- **Interactive Mode**: Prompts for user input and provides personalized greetings
- **Clean Code**: Well-documented with docstrings and comments
