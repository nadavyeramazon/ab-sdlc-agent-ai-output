# Hello World Frontend

A minimal React application with green theme and backend integration.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start development server:
   ```bash
   npm run dev
   ```

3. Open browser at: http://localhost:3000

## Features

- Static "Hello World" heading with green theme
- Button to fetch data from backend API
- Loading state indicator
- Error handling and display
- Responsive design

## Tech Stack

- React 18.2.0
- Vite 5.0.0
- Native Fetch API (no axios)
- Simple CSS (no framework)

## Color Scheme

- Primary green: #2ecc71
- Secondary green: #27ae60
- White text on green: #ffffff
- Background: light gray (#f5f5f5)

## Backend Integration

The app connects to backend API at `http://localhost:8000/api/hello`

Make sure the backend server is running before testing the integration.
